Descrição do projeto

O projeto sera um editor online para codigo c#, o usuario podera editar seu codigo no front-end e envia-lo para o back-end para executar usando tecnologias com runtime compiler, o back-end retornara o resultado do programa entao mostrando no campo log.

Recursos usados no projeto

Um servidor Linux de baixo desempenho com 1 core e 1gb de ram
